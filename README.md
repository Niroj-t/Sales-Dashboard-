# 📊 Sales & Profit Dashboard

## 📌 Project Overview

This project is a data visualization dashboard designed to analyze **sales and profit performance from 2014 to 2017**. It provides a clear and interactive view of business performance across categories, time periods, and geographical locations. The goal is to help users quickly understand key business trends, identify top-performing segments, and make data-driven decisions.

---

## ✨ Feature Breakdown

### 📦 Sales by Category (Funnel Chart)
- Displays sales distribution across different product categories.
- Highlights **Storage** as the highest-selling category with **$58,788.72** in sales.
- Helps identify which categories contribute most to revenue.

### 📈 Profit Gained Over Time (Line Chart)
- Shows profit trends from **2014 to 2017**.
- Helps track business growth and seasonal fluctuations.
- Useful for identifying high and low-performing periods.

### 🗺️ Sales by State (Map Visualization)
- Interactive map showing sales distribution across different states.
- Helps visualize geographical performance and regional strengths.
- Supports location-based decision making.

### 📊 Monthly Sales (Area Chart)
- Displays monthly sales trends over time.
- Helps identify seasonal demand patterns.
- Useful for forecasting and planning future sales strategies.

### 👥 Top Customers
- Displays the **top 5 customers** contributing the most revenue.
- Helps in identifying key customer relationships and retention opportunities.

---

## 💡 Data Insights

- 📌 **Highest Sales Category:** Storage – **$58,788.72**
- 📌 Strong sales concentration in top-performing categories.
- 📌 Clear upward and fluctuating profit trends across 2014–2017.
- 📌 Key customers contribute significantly to total revenue.
- 📌 Regional performance varies across different states.

## 🛠️ Tech Stack

This dashboard can be implemented using modern web and data visualization technologies:

- **Frontend Framework:** Next.js
- **Styling:** Tailwind CSS
- **Charts & Visualization:** Recharts / Chart.js / D3.js
- **Data Handling:** REST API / JSON / Excel dataset integration

## 🚀 Goal

To build an interactive, responsive, and insightful analytics dashboard that transforms raw sales data into meaningful business intelligence.
